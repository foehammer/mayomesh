Mayo Mesh Observer Setup

Set your device name and Wi-Fi credentials over USB — no serial terminal or typed commands required.

Browser not supported

This tool uses the Web Serial API, which only works in Chrome or Edge (desktop). Firefox and Safari aren't supported. If you're already in Chrome/Edge, make sure you're on https:// (not opening the file locally).

You can always fall back to a serial terminal and the manual commands in the MQTT setup guide.
